# Thai ID Card Agent — โปรแกรมอ่านบัตรประชาชน

โปรแกรมตัวเล็กที่รันบนเครื่องผู้ใช้ คุยกับเครื่องอ่านบัตร (PC/SC) แล้วเปิด HTTP
server บน `127.0.0.1:8765` ให้เว็บแอป (Google Apps Script) ดึงข้อมูลบัตรได้

## ทำไมต้องมี agent
เครื่องอ่านบัตรประชาชนเป็นอุปกรณ์ **PC/SC (USB CCID)** ซึ่งเบราว์เซอร์คุยตรงไม่ได้
จึงต้องมีโปรแกรมตัวกลางบนเครื่อง อ่านบัตรแล้วส่งให้หน้าเว็บผ่าน localhost

## ความต้องการ
- Windows + Python 3.x
- เครื่องอ่านบัตร smart card (PC/SC) เสียบ USB
- ไลบรารี `pyscard` (สคริปต์ติดตั้งให้อัตโนมัติ)

## วิธีใช้
1. เสียบเครื่องอ่านบัตร และเสียบบัตรประชาชน
2. ดับเบิลคลิก **`start-agent.bat`** (ครั้งแรกจะติดตั้ง pyscard ให้เอง)
3. เปิดเว็บแอป → กดปุ่ม **"อ่านบัตรประชาชน"** → ข้อมูลจะถูกเติมให้อัตโนมัติ

> ต้องเปิด `start-agent.bat` ค้างไว้ระหว่างใช้งาน (ปิดหน้าต่างนี้ = ปิด agent)

## รันด้วยมือ (ถ้าไม่ใช้ .bat)
```
pip install pyscard
python idcard_agent.py
```

## Endpoints
| Method | Path | ผลลัพธ์ |
|--------|------|---------|
| GET | `/status` | สถานะเครื่องอ่าน/บัตร |
| GET | `/read` | ข้อมูลบนบัตร (cid, ชื่อ, ที่อยู่ ฯลฯ) |

## หมายเหตุ
- **รหัสไปรษณีย์ไม่มีบนบัตร** ต้องกรอกเอง
- ข้อความบนบัตรเข้ารหัส TIS-620 — โปรแกรมแปลงเป็น UTF-8 ให้แล้ว
- เบราว์เซอร์ที่รองรับการเรียก `http://127.0.0.1` จากหน้า HTTPS: Chrome/Edge
  (มี header `Access-Control-Allow-Private-Network` รองรับ Chrome รุ่นใหม่)
- ถ้ากดปุ่มแล้วขึ้น "เชื่อมต่อโปรแกรมอ่านบัตรไม่ได้" = ยังไม่ได้เปิด `start-agent.bat`
